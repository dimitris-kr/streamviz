Project Development

The visualizations included in this website were created using Tableau Desktop, were uploaded to Tableau Public and can be accessed there by clicking the corresponding links in each page containing a visualization.

The website itself was developed using the Angular web framework and the project's full source code can be found on GitHub. In addition, the GitHub Pages service was utilized to host the website permanently.

GitHub Repository .

Datasets

We tried to include as many and as recently updated data as possible, so we combined various sources to create a single .xlsx file following the relational model. The statistics (subscribers, revenue) included are up to the most recent completed quarter, which when the project was conducted was Q1 of 2023. While, the statistics are based on worldwide date, the content availability and monthly fee are region-specific for the United States. The movies and series's availability on Disney+, HBO Max, Netflix and Prime Video, was updated in March 2023, while on HULU it was in May 2022. The currency in every visualization is US dollars.
